# vlsp2023

## Set up

**Create a new virtual enviroment and activate it**

```bash
    python -m venv venv
    .\venv\Scipts\activate.bat
```
**Install requirements**

```bash
    pip install -r requirements.txt
```

## Demo

**Backend**
```bash
    cd backend
    python main.py
```
**Frontend**
```bash
    cd frontend
    python [task1/task2].py
```
